Le code est en Python et utilise la bibliothèque Turtle pour dessiner des figures géométriques.

Le code commence par importer les bibliothèques Turtle . Ensuite, une nouvelle fenêtre graphique est créée en utilisant la méthode Screen() de la bibliothèque Turtle. La taille de l'écran est définie en utilisant la méthode screensize().

La couleur de fond de l'écran est définie en utilisant la méthode bgcolor() de la bibliothèque Turtle.

Ensuite, un carré est dessiné à l'aide de la méthode forward() et left() qui permettent de déplacer le crayon.

Après le carré, un parallélogramme et un triangle sont dessinés en utilisant les mêmes méthodes.

Enfin, la méthode mainloop() est appelée pour maintenir l'écran ouvert.



Voici une procédure générale qui peut vous aider:

   Téléchargez et installez Python sur votre ordinateur si vous ne l'avez pas déjà fait. Vous pouvez télécharger Python sur le site officiel de Python : https://www.python.org/downloads/

Ouvrez une invite de commande ou un terminal et naviguez jusqu'au répertoire où se trouve votre fichier Python.

Exécutez le fichier Python en tapant "python3 figure_graphique.py " sous linux et 'python figure_graphique.py' sous windows dans l'invite de commande ou le terminal et appuyez sur Entrée. Assurez-vous que Python est bien installé et configuré sur votre système pour que cela fonctionne.

Le fichier Python s'exécutera et produira une sortie en fonction de ce que vous avez codé.
